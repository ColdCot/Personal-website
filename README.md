# Simple ASP.NET Website Backend

Backend for my personal website (frontend coming soon). This is a simple training project — don't take it too seriously!

## 🚀 Features

*   **Message Management**: Receive messages and store them in a database.
*   **Admin Authentication**: Secure login for admins to read user messages.
*   **Sender Control**: Create and delete senders as needed.

## 🛠 Tech Stack

*   **Framework**: ASP.NET Core
*   **Database**: Entity Framework Core (SQL Server)
*   **Auth**: JWT-based Authentication
